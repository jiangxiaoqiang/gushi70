## 小学生必背古诗70首

为方便广大家长随时随地虐孩子，特意编辑了《小学生必背古诗70首》，按照手机尺寸编辑，放大了字体，方便阅读。为了认几个字，总不能让娃娃们把眼睛搞瞎了吧。

另外还对诗词进行了注音，因编辑知识有限，发音不标准，难免有错误。。PS，我从小学开始，拼音就没考及格过:(。如发现有错误请反馈至yangjianlin@gmail.com，我们会进行修正。

最新版请到这里自行下载，恕不一一通知。
